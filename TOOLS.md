# TOOLS.md - Local Notes

## 📂 输入/输出

| 目录 | 用途 | 写入权限 |
|:----|:-----|:--------|
| `input/` | Hans 放置资料（法规/平台规则/供应商信息等） | Hans（你）→ 我读取 |
| `output/` | 我生成的方案/报告/文件 | 我写入 → Hans 取用 |

### input/ 子目录

| 目录 | 放什么 |
|:----|:------|
| `法规文件/` | 法律法规原文、平台公告截图、合规文件 |
| `平台规则/` | 各平台商家规则、处罚细则、禁词表 |
| `供应商资料/` | 1688等供应商信息、报价单、资质文件 |
| `商品资料/` | 产品参数、质检报告、实拍图 |
| `运营数据/` | 经营报表、订单数据、广告数据 |
| `配置/` | API密钥配置文件、底层参数配置 |
| `其他/` | 未归类的文件 |
