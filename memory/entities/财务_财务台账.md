# [[财务_财务台账]]

## 属性
- 类型: 财务 | 台账
- 状态: 待激活（等待店铺有订单后自动记账）

## 记账维度
| 维度 | 说明 |
|:----|:----|
| 订单编号 | 每个订单唯一标识 |
| 商品SKU | 商品规格 |
| 售价 | 客户实付 |
| 进价 | 供应商采购价 |
| 物流费 | 供应商发货运费 |
| 平台佣金 | 平台扣点 |
| 退货成本 | 退换货损耗 |
| 广告费 | 单品广告支出 |
| 毛利润 | 售价-进价-物流-佣金 |
| 毛利率 | 毛利润/售价 |

## 报表体系
- 小时报：实时GMV、订单数、广告消耗
- 日报：昨日营收、利润、退款、差评
- 周报：本周利润、投产、售后复盘
- 月报：月度财务台账、品类盈亏分析
