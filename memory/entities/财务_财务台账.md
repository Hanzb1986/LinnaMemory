# [[财务_财务台账]]

## 属性
- 类型: 财务 | 台账
- 模块归属: 企业财税体系
- 状态: 待激活

## 记账维度
| 维度 | 说明 | 数据来源 |
|:----|:-----|:--------|
| 订单编号 | 每个订单唯一标识 | 业务模块 |
| 商品SKU | 商品规格 | 产品模块 |
| 售价 | 客户实付 | 业务模块 |
| 进价 | 供应商采购价 | 产品模块/BOM |
| 物流费 | 供应商发货运费 | 业务模块 |
| 平台佣金 | 平台扣点 | 业务模块 |
| 退货成本 | 退换货损耗 | 质量模块(售后) |
| 广告费 | 单品广告支出 | 业务模块 |
| 人力成本 | 薪酬社保 | 人资模块 |
| 质检损耗 | 瑕疵品成本 | 质量模块 |
| 毛利润 | 售价-成本合计 | 自动计算 |
| 税前扣除 | 合规抵扣项 | 各模块汇总 |

## 报表体系
- 小时报：实时GMV、订单数、广告消耗
- 日报：昨日营收、利润、退款、差评
- 周报：本周利润、投产、售后复盘
- 月报：月度财务台账、品类盈亏分析
