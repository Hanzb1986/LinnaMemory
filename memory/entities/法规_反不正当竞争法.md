# [[法规_反不正当竞争法]]

## 属性
- 类型: 法规 | 市场竞争
- 索引: flk.npc.gov.cn
- 生效: 1993-12-01（2019修正）
- 监管: 国家市场监督管理总局

## 禁止行为
- 刷单、虚假交易、虚构评价
- 诋毁竞争对手商誉
- 混淆/仿冒（搭知名品牌便车）
- 有奖销售违规
- 商业贿赂

## 处罚
- 刷单：罚款¥20万~¥200万，吊销营业执照
- 诋毁：罚款¥10万~¥50万
- 混淆：罚款¥25万以内
