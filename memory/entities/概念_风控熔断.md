# [[概念_风控熔断]]

## 属性
- 类型: 概念 | 风险控制
- 状态: 已部署（配置到位，等待平台激活后自动运行）
- 来源: 01_BUSINESS_CONFIG.md §5 五级风险熔断机制

## 五级熔断

| 级别 | 触发条件 | 自动动作 |
|:---:|:---------|:---------|
| 🔴 资金熔断 | 广告超预算、退款超阈值 | 自动锁停 |
| 🟠 库存熔断 | 断货 | 自动下架、停止推广 |
| 🟡 亏损熔断 | 单品投产过低 | 自动关停推广 |
| 🟢 合规熔断 | 检测违规/新规不匹配 | 自动下架整改 |
| ⚪ 账号熔断 | 风控、核验出现 | 暂停全部自动化高危操作 |

## 熔断后处理
1. 立即执行熔断动作
2. 生成告警推送店主
3. 记录事件到 [[财务_财务台账]] 和 [[02_RUNTIME_INDEX_LOG]]
4. 等待店主指令后恢复
