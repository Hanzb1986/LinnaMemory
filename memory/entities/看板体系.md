## 四层分层看板

### ① 一线操作看板（实时异常监控）
- Agent同步失败订单 / 待处理售后 / 低库存预警
- 目标：快速定位自动化故障，减少人工兜底

### ② 运营增长看板（2小时聚合）
- SKU价值分层 / 广告投产 / 多店对比 / AI预估ROI vs 实际偏差
- 目标：反向优化AI测算模型

### ③ 财务风控看板（日终汇总）
- 资金审批监控 / 对账差异 / 证照到期 / 线上营收+线下费用分区
- 目标：资金风险管控

### ④ 管理层总览看板（日度汇总）
- 核心经营指标 / 全店亏损汇总 / 大额资金 / 高危预警
- 目标：极简，屏蔽细碎数据

## 看板安全
- 单店人员仅看本店，多店总览仅管理层
- 只读属性，不可修改原始数据
- 导出/筛选/查看明细均记录账号/时间

## 自动分发
- 每4小时推送精简摘要
- 24点推送全日日报
- 红色预警实时推送
- 回溯存储：历史日看板永久归档
