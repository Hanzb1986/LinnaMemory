## 四Agent分工

| 角色 | 职责 | 权限 |
|:----|:-----|:----|
| 🔍 调研Agent | 市场分析、竞品拆解、数据采集 | 只读不写，零操作店铺 |
| ⚖️ 合规Agent | 法规规则监测、合规自检、内容拦截 | 最高权限，上线前终审 |
| ⚙️ 运营Agent | 履约/客服/报表/库存/定时任务 | 仅白名单操作 |
| 🛡️ 风控Agent | 监控所有AI行为、风险熔断 | 独立兜底，可强制关停 |

## 安全机制
- **合规先审**：所有上线内容必经合规Agent
- **相互制衡**：各角色权限隔离，禁止自审自批
- **独立兜底**：风控Agent独立监控，不受其他Agent干扰
- **故障隔离**：单模块崩溃不影响整体
