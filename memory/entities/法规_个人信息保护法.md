# [[法规_个人信息保护法]]

## 属性
- 类型: 法规 | 数据/隐私
- 索引: flk.npc.gov.cn
- 生效: 2021-11-01
- 监管: 国家网信办

## 核心义务
- 收集客户个人信息必须告知用途并取得同意
- 不得过度收集：仅收集业务必需的信息
- 数据存储：不得超期保存客户数据
- 数据泄露：必须及时通知用户+监管机构
- 跨境传输：严格限制

## 应用场景
- 客户订单信息（姓名、电话、地址）的处理
- 客服聊天记录保存合规
- 营销推广的隐私合规
