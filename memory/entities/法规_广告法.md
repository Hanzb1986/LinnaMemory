# [[法规_广告法]]

## 属性
- 类型: 法规 | 广告合规
- 索引: flk.npc.gov.cn
- 生效: 2015-09-01（2018修正）
- 监管: 国家市场监督管理总局

## 禁止行为
- 使用极限词：国家级、最高级、最佳、第一等
- 虚假宣传：与实际功能不符的描述
- 医疗/食品特殊宣传：未经审批不得宣传功效
- 损害未成年人身心健康

## 应用场景
- 商品标题、主图文案、详情描述
- 直播话术、短视频文案
- 广告投放物料

## 处罚
- 罚款¥20万~¥100万
- 情节严重的吊销营业执照
- 广告费用无法计算时，罚款¥20万~¥200万
