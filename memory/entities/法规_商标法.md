# [[法规_商标法]]

## 属性
- 类型: 法规 | 知识产权
- 索引: flk.npc.gov.cn
- 生效: 1983-03-01（2019修正→2026修订）
- 最新修订: 2026-06-26（主席令第77号）
- 最新施行: 2027-01-01
- 监管: 国家知识产权局

## 禁止行为
- 销售假冒注册商标商品
- 未经授权使用他人商标
- 傍名牌（近似商标混淆）
- 误导性商标使用（2026修订加重处罚）

## 处罚
- 假冒注册商标罪：刑事追诉
- 商标侵权赔偿：权利人实际损失或违法所得
- 误导性商标使用行为惩戒力度加大（2026修订版）

## 2026修订要点
- 通过时间：2026-06-26，十四届全国人大常委会第二十三次会议
- 施行时间：2027-01-01
- 核心变化：强化商标管理和保护；避免误伤防御性注册商标；对误导性商标使用加大惩戒力度
- 来源：http://www.npc.gov.cn/npc/c2/kgfb/202606/t20260626_455859.html

## 应用场景
- 选品：确认商品不侵犯他人商标权
- 店铺名称/品牌名称：不蹭大牌
- 主图/描述：不使用他人注册商标
