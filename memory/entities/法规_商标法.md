# [[法规_商标法]]

## 属性
- 类型: 法规 | 知识产权
- 索引: flk.npc.gov.cn
- 生效: 1983-03-01（2019修正）
- 监管: 国家知识产权局

## 禁止行为
- 销售假冒注册商标商品
- 未经授权使用他人商标
- 傍名牌（近似商标混淆）

## 处罚
- 假冒注册商标罪：刑事追诉
- 商标侵权赔偿：权利人实际损失或违法所得

## 应用场景
- 选品：确认商品不侵犯他人商标权
- 店铺名称/品牌名称：不蹭大牌
- 主图/描述：不使用他人注册商标
