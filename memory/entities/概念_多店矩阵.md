# [[概念_多店矩阵]]

## 属性
- 类型: 概念 | 运营策略
- 状态: 已入库（运营规则就绪，待实际跑通后扩展）

## 核心原则
1. **不存在无限开店**：四层硬天花板（平台限制/资质上限/风控关联/资金人力）
2. **主体隔离**：不同营业执照独立运营组
3. **关联组管理**：同法人下店铺划为「关联组」
4. **三级熔断**：单品→单店→集群

## 实施节奏
1. **起步**：1-3家同主体，跑通验证
2. **稳定后扩店**：个体执照≤3家，企业执照≤10家
3. **多主体矩阵**：每家独立法人5-8家店
