## 业务流程
```
[选品] → [上架] → [接单] → [采购] → [发货] → [记账]
                ↑                    ↕
           自动上架              货源同步
```

## 核心特征
- 零库存：不上库存，客户下单后同步采购
- 零打包：供应商直接发货给客户
- 零发货：物流回填自动完成
- 利润 = 售价 - 进价 - 物流费 - 平台费 - 退换货损耗

## 关键节点
| 环节 | 自动化程度 | 备注 |
|:----|:----------|:----|
| 选品 | 半自动 | AI筛选+店主审核（黑名单操作） |
| 上架 | AI生成方案→店主审核 | 禁止AI自主上新 |
| 接单 | 全自动 | 无需人工 |
| 采购 | 首次确认后自动 | 需先经Hans授权 |
| 发货 | 全自动（物流回填） | 供应商直发 |
| 记账 | 全自动 | 每单进价/售价/运费/利润 |
