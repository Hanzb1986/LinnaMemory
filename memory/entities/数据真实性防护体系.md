# [[数据真实性防护体系]]

## 属性
- 类型: 概念 | 系统底线
- 来源: 2026-06-29 Hans 定稿
- 优先级: 高于一切，不可突破

## 零篡改底线
全链路原始底层数据一律只读，不可手动编辑。禁止任何角色修改原始业务数据。

### 原始数据 vs 测算数据
| 类型 | 定义 | 可操作性 |
|:----|:-----|:--------|
| 🔒 原始数据 | 订单/报价单/物流/银行/发票/售后工单 | 只读，不可编辑 |
| 📊 测算数据 | 毛利率/ROI/预测销量/预估损耗 | 标注「测算参考」，不替代真实数据 |

## 三套自动对账（财税岗每日执行）
| 对账 | 公式 | 不平则预警 |
|:----|:-----|:----------|
| 业务对账 | 平台订单销售额 = 履约订单台账合计 | ✅ |
| 财务对账 | 订单收款 - 退款 - 平台扣费 = 银行流水 | ✅ |
| 供应链对账 | 入库数量 - 退货数量 - 售出数量 = 当前库存 | ✅ |

## 三级数据标签（所有报表强制标注）

### 🟢 一级·原生可信数据
- 来源：API同步/银行流水/完税凭证/平台导出原件/盖章报价扫描件/质检原始记录
- 水印：【原生可信·自动同步数据源】
- 使用：对账/税务/真实盈亏判定/审计——以此为准
- 存档：`output/origin_data/`（加密只读）

### 🟡 二级·人工补充数据
- 来源：手动上传报价/广告账单/物流对账单
- 水印：【人工补充·存在录入误差，建议核对原始单据】
- 使用：单独分区展示，不单独作最终结论依据
- 存档：`output/manual_upload/`（留存原件，不可删除）

### ⚪ 三级·测算预估数据
- 来源：AI预估毛利/ROI/预测销量/库存损耗预判
- 水印：【测算预估·仅供方案对比，真实盈亏以原生数据为准】
- 使用：审批单/选品方案/投放规划内对比参考，复盘报告分栏隔离
- 存档：`output/analysis/`（推演文件，仅参考）

### 混合数据文档排版标准
- 多级数据同表：分三栏独立展示（绿/黄/灰），栏头标等级，禁止同列混杂
- 审批单底部：《数据来源说明栏》逐条列明测算数据对应级别
- 周/月报固定：数据可信度小结板块（一级条数占比 / 二级缺失清单 / 三级参考项）
- 差异分类标注：一级数据冲突、二级录入误差、三级预估偏差——各自标注处理方式

### 特殊场景规则
- 一级数据缺失仅靠二级核算：全报表红色警示顶置，禁止输出最终盈利结论
- 三级测算与一级历史偏差超20%：高亮标黄，附加风险提示
- 调整分录：标记为二级，绑定一级原始差异单据编号留痕

### 违规判定（触碰即数据风险）
1. 未标注数据等级直接输出报表
2. 三级测算不做隔离，直接与一级数据合并出结论
3. 二级/三级数据单独作为报税/对账/审计依据
4. 去除/遮盖/修改文件自带分级水印

## 权限隔离
- 选品岗：仅选品方案+评级，无权修改订单/财务
- 履约岗：仅订单库存读写，无权修改实付金额
- 风控岗：仅新增售后记录，无权改退款金额
- 财税岗：仅归集核算，无权改原始业务单据
- 你：仅备注差异说明，不直接改原始数字

## 历史数据锁定
- 月度/年度结束后自动归档加密
- 不可编辑归档文件（md+加密Excel）
- 调整仅能在当期新增调整分录

## 误差 vs 造假处置
| 场景 | 处置 |
|:----|:-----|
| 非主观失真（漏传/算错/时差） | 补充凭证→新增调整记录留痕 |
| 主观造假（改金额/删记录/虚构） | 锁定操作→暂停权限→全局告警→永久留存 |

## 数据安全联动规则（分级绑定权限/加密/脱敏/删除）

### 一级·原生可信（最高安全）
| 维度 | 规则 |
|:----|:-----|
| 可读 | 财税岗+股东全量；其他AI仅脱敏副本 |
| 可写 | 全角色禁止编辑/删除/覆盖，永久只读 |
| 加密 | `output/origin_data/` 最高加密，双重备份 |
| 留存 | 永久保存 |
| 删除 | 系统强制拦截，触碰即红线事故 |

### 二级·人工补充（中等安全）
| 维度 | 规则 |
|:----|:-----|
| 可读 | 全部AI+股东；客服无权限 |
| 可写 | 仅股东可上传更新，AI只读 |
| 加密 | `output/manual_upload/` 基础加密 |
| 留存 | 3年归档封存 |
| 删除 | 需审批单+股东批准，日志留存 |

### 三级·测算预估（基础安全）
| 维度 | 规则 |
|:----|:-----|
| 可读 | 全角色可读 |
| 可写 | 全AI可生成/更新 |
| 加密 | `output/analysis/` 无高强度加密 |
| 留存 | 1年自动清除 |
| 删除 | 自动过期清理，手动删除仅记日志 |

### 自动分配机制
- 文件生成→自动打分级标签→瞬间匹配安全规则
- 标签被移除→自动下调防护+全局告警
- 跨角色传送带标签→客服收一级文件自动脱敏成本/利润字段

### 违规分级处置
| 等级 | 行为 | 处置 |
|:----|:-----|:-----|
| 🔴 一级 | 私自导出/篡改/删除原始凭证 | 冻结权限+红线事故+股东当日复核 |
| 🟡 二级 | 无审批删除上传单据 | 风险记录+周度核查+限期补交 |
| ⚪ 三级 | 随意删除测算报表 | 仅记日志 |
