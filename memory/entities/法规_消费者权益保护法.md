# [[法规_消费者权益保护法]]

## 属性
- 类型: 法规 | 售后/维权
- 索引: flk.npc.gov.cn
- 生效: 2014-03-15（修正版）
- 监管: 国家市场监督管理总局

## 核心条款
- **七天无理由退货**：网购商品适用，定制/鲜活/音像制品除外
- **欺诈三倍赔偿**：退一赔三，最低¥500
- **质量三包**：修理、更换、退货
- **知情权**：商品真实信息必须完整披露
- **隐私权**：消费者个人信息受保护

## 应用场景
- 客服退货话术
- 售后纠纷处理
- 12315投诉应对
