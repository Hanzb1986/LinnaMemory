# [[法规_电子商务法]]

## 属性
- 类型: 法规 | 核心电商法律
- 索引: flk.npc.gov.cn
- 生效: 2019-01-01
- 监管: 国家市场监督管理总局
- 适用范围: 所有线上开店经营行为

## 义务要求
- 主体登记：办理营业执照
- 公示信息：店铺首页展示营业执照信息
- 交易记录：保存≥3年
- 知识产权：平台有保护义务，商家不得侵权
- 禁止：虚假交易、刷单、恶意评价

## 禁止行为
- 刷单炒信、虚假销量
- 默认搭售
- 大数据杀熟
- 恶意拦截/屏蔽

## 处罚
- 罚款：最高¥200万
- 情节严重的吊销营业执照
