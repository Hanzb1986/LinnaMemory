# [[质量管控]]

## 属性
- 类型: 质检/品控/售后
- 模块归属: 质量管控体系
- 覆盖: 质检标准/入库质检/出库抽检/售后质量溯源/客诉统计/问题汇总/整改

## 全链路质量管控流程
```
供应商来料 → 入库质检（全检/抽检）→ 仓储巡检（保质期/环境）
  → 出库质检（复核）→ 客户收货
  → 售后质量（退换货/客诉）→ 问题汇总 → 溯源 → 整改
```

## 质量台账体系
| 台账 | 记录内容 | 更新频次 |
|:----|:---------|:--------|
| 入库质检台账.xlsx | 批次/到货数量/合格率/瑕疵明细 | 每次入库 |
| 仓储巡检台账.xlsx | 环境/保质期/破损/虫害 | 日/周 |
| 出库质检台账.xlsx | 复核结果/错漏/包装完好率 | 每次出库 |
| 售后质量问题台账 | 退换货原因/客诉分类/赔付金额 | 实时 |
| 客诉统计台账 | 投诉类型/频率/处理结果/满意度 | 日/周/月 |
| 问题品整改台账 | 批次/问题描述/整改措施/闭环记录 | 整改后 |
| 质量风险预警 | 高频问题/批次趋势/供应商评级 | 周/月 |

## 客诉与售后质量问题闭环
```
客户投诉/退换货
  → 质量问题分类（破损/错发/缺件/质量瑕疵/虚假描述）
  → 责任判定（供应商/仓储/物流/售前误导）
  → 赔付处理（小额自动/大额人工终审）
  → 问题录入台账
  → 溯源分析（批次/供应商/操作环节）
  → 整改方案（供应商更换/流程优化/培训）
  → 效果验证
```

## 退货质检分级标准
| 等级 | 定义 | 处理路径 |
|:----|:-----|:--------|
| 🟢 可二次销售 | 包装完好/商品无损 | 重新入库 |
| 🟡 需翻新 | 轻微瑕疵/可修复 | 翻新处理→质检→可售/报废 |
| 🔴 报废退回供应商 | 严重瑕疵/过期 | 报废→供应商索赔/冲账 |

## 联动机制
- **→ 产品模块**: 高频质量问题反馈→产品选型优化/资质审查
- **→ 业务模块**: 不合格品锁定库存→禁止发货；售后数据同步业务台账
- **→ 财税模块**: 赔付成本→税前扣除；质检损耗→成本核算
- **→ 人力模块**: 质检人员培训需求→岗位合规
