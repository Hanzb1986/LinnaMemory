# [[法规_标准化法]]

## 属性
- 类型: 法规 | 标准体系
- 索引: flk.npc.gov.cn
- 生效: 2018-01-01（修订版）
- 监管: 国家市场监督管理总局 / 国家标准化管理委员会

## 核心要求
- 产品须符合国家标准（GB）/ 行业标准（QB/T等）
- 无标准产品禁止生产/销售
- 鼓励企业制定高于国标的企业标准

## 应用场景
- 选品：确认产品有执行标准
- 商品详情：标准号标注
- 质检合规
