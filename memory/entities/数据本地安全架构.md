# [[数据本地安全架构]]

## 属性
- 类型: 概念 | 安全架构
- 来源: 2026-06-29 Hans 定稿
- 核心: 原始数据本地加密留存，云端仅收脱敏推演请求

## 铁律
1. 禁止向云端模型传输任何**一级原生可信数据**（订单/报价/银行/发票/客户/成本）
2. 所有真实核算、对账、记账在本地完成，不依赖云端返回值
3. 本地加密文件不上传任何云端存储

## 模型调用前脱敏规则
| 数据类型 | 传云端内容 | 禁止传输 |
|:--------|:----------|:---------|
| 金额 | 比例/区间 | 真实进价/售价 |
| 商品 | 品类/规格（去成本） | 采购成本/供应商名 |
| 客户 | 掩码信息 | 手机号/姓名/地址 |
| 财务 | 汇总比例 | 单笔流水 |

## 本地安全
- `local_data/` 根目录，3层加密子目录
- 一级：AES-256加密，独立密钥仅你持有
- 本地哈希校验码防篡改
- 每日本地双硬盘快照 + 月度U盘异地归档
