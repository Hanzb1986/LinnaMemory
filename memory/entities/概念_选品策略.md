## 选品原则
1. **利润优先**：毛利率≥店铺设定底线，低毛利不碰
2. **数据驱动**：看销量趋势、竞品分析、搜索热度，不看感觉
3. **风险过滤**：假货/侵权/违禁品类不碰
4. **售后可控**：退换货率高的品类需额外评估

## 选品流程
1. 供应商端爬取/API获取商品数据（1688等）
2. 利润测算：进价+运费+平台费+退货成本 vs 售价
3. 合规预审：匹配广告法/电商法/产品质量法
4. 竞争分析：同款竞品价格/销量/评价
5. 生成《上新方案》→ 提交店主审核
