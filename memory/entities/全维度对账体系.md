## 每日自动对账（当日完成，差异当日预警）
| 对账 | 校验公式 | 异常处置 |
|:----|:---------|:--------|
| 📊 业务订单对账 | 平台API订单销售额 = 履约订单台账合计 | 差异清单推审核池 |
| 💰 财务资金对账 | 订单收款-退款-平台扣费 = 银行账户流水 | 无审批支出标红 |
| 📦 供应链库存对账 | 期初+采购-售出-报废 = 当前账面库存 | 盘盈盘亏推审核池 |

## 每周复盘对账
| 对账 | 校验内容 |
|:----|:---------|
| 利润对账 | 单品净利润累加 = 店铺整体净利润 |
| 投放对账 | 广告后台消耗 = 财税广告分摊台账 |
| 质量对账 | 售后赔付总额 = 财务损耗科目 |

## 月度专项对账（月末5个工作日）
| 对账 | 内容 |
|:----|:-----|
| 税务对账 | 营收≈开票+未开票收入；进项≈采购台账；税负波动排查 |
| 资金预算对账 | 计划审批总预算 vs 实际总支出，刚性80%/弹性70%预警 |
| 库存全盘对账 | 全SKU账面 vs 理论库存，汇总盘亏盘盈及损耗原因 |
| 合规对账 | 在售SKU vs 有效资质台账，无资质商品标记下架 |

## 年度封存审计
全年订单/财务/库存/资金审批/完税凭证统一封存

## 对账统一处置流程
1. 差异→自动生成《对账差异清单》（金额/单据/成因）
2. 推送股东审核池
3. 补充凭证/实物盘点后→当期新增调整分录→留痕不删原始记录
4. 月度对账记录归档永久留存
