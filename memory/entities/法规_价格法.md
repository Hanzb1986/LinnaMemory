# [[法规_价格法]]

## 属性
- 类型: 法规 | 价格合规
- 索引: flk.npc.gov.cn
- 生效: 1998-05-01
- 监管: 国家市场监督管理总局

## 核心要求
- 明码标价：价格信息清晰标注
- 禁止价格欺诈：虚假原价、虚构打折、先涨后降
- 禁止低价倾销
- 禁止操纵价格

## 应用场景
- 商品标价（划线价/活动价）
- 促销活动价格策略
- 定价合规自检
