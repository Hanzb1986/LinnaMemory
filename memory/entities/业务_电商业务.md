```
                     ┌──────────────┐

                     └──────┬───────┘
                            │
┌──────────┐   ┌──────────┐│┌──────────┐   ┌──────────┐   ┌──────────┐
│  选品    │ → │  上架    │││  接单    │ → │  采购    │ → │  发货    │
│ (业务)   │   │ (运营)   │││ (自动)   │   │ (供应商)  │   │ (物流)   │
└────┬─────┘   └────┬─────┘│└────┬─────┘   └────┬─────┘   └────┬─────┘
     │              │      │     │              │              │
     ▼              ▼      │     ▼              ▼              ▼
```

## 各环节交叉关联（全模块版）

| 环节 | 业务动作 | → 产品模块 | → 质量模块 | → 财税模块 | → 人力模块 | → 法务 |
|:----|:--------|:-----------|:-----------|:-----------|:-----------|:------|
| 选品 | 供应商谈判 | BOM成本/资质预审 | 供应商质量评级 | 成本→增值税抵扣 | — | 供货合同 |
| 上架 | 商品发布 | SKU参数/SKU编码 | 上架前质检标准 | 定价→毛利计算 | 运营人员配置 | 广告法合规 |
| 接单 | 订单处理 | 库存扣减 | 发货质检触发 | 收入确认时点 | 客服排班 | 民法典合同成立 |
| 采购 | 供应商下单 | BOM物料/编码匹配 | 入库质检触发 | 成本发票取得 | 采购人员 | 采购合同 |
| 发货 | 物流配送 | 库存出库 | 出库抽检复核 | 运费成本 | 仓管/打包 | 货损追偿 |
| 售后 | 退换货/投诉 | 产品问题登记 | 质量问题溯源台账 | 退款税务处理 | 客服培训 | 消保法 |
| 记账 | 财务核算 | 产品成本归集 | 质检损耗核算 | 全部税种申报 | 薪酬成本核算 | 档案留存 |

## 跨模块数据流向

```
业务订单量 → 人力排班预测
业务营收 → 财税收入核算
业务库销 → 产品生命周期判定
业务售后 → 质量客诉统计
产品BOM → 业务定价/采购成本
产品资质 → 业务上架合规
质量客诉 → 业务赔付处理
质量品控 → 业务供应商评级
人力绩效 → 业务效率分析
财税成本 → 业务利润测算
```
