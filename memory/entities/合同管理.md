# [[合同管理]]

## 属性
- 类型: 法务/业务
- 适用法律: 民法典合同编 + 电子签名法

## 常用合同类型

| 合同类型 | 适用场景 | 关键条款 |
|:--------|:--------|:---------|
| 电商交易合同 | 客户下单（电子合同） | 民法典第491条：提交订单即成立 |
| 供货/代发合同 | 供应商合作 | 质量/交货/售后/违约责任 |
| 劳动合同 | 雇佣员工 | 试用期/薪酬/社保/解除条件 |
| 保密协议 | 员工/供应商 | 商业秘密/竞业限制（≤2年） |
| 租赁合同 | 仓储/办公室 | 民法典合同编租赁合同章节 |
| 服务协议 | 广告/推广合作 | 服务内容/费用/违约 |

## 电子合同合规
- 可靠电子签名与手写签名具有同等法律效力（电子签名法第14条）
- 电子合同成立时间：提交订单成功时合同成立（民法典第491条）
- 格式条款：须显著提示+公平合理（民法典第496-498条）
