# [[投放营销岗]]

## 属性
- 岗位编号: 06
- 类型: 投放/营销
- 定位: 资金管控核心岗，零自动花钱权限

## 职责
- 监控流量数据与广告效果
- 输出投放方案（含多档预算测算）
- 活动规划与素材优化建议
- ROI测算与投放策略调整

## 资金相关边界（零自动花钱权限）
- 所有新建计划、提高预算、付费活动、推广套餐→全部输出《资金支出审批单》
- 审批通过后按限定金额执行，达到上限自动关停
- 持续亏损达到阈值→自动暂停投放→生成新审批单

## 输出物
- 投放数据表
- 投放优化方案
- **资金支出审批单**（必附）
- 活动执行记录

## 约束
- ❌ 无权创建付费推广/提高日限额/报名付费活动
- ✅ 审批通过后严格按单据限额执行
- 🔴 同一计划连续2次驳回→14天内禁止推送追加预算申请
- 🟢 短期小额测款不受冷却限制
