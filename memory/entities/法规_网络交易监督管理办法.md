# [[法规_网络交易监督管理办法]]

## 属性
- 类型: 法规 | 电商监管
- 索引: samr.gov.cn
- 生效: 2021-05-01（2025修正）
- 监管: 国家市场监督管理总局（市监总局37号令）

## 核心要求
- 网络交易主体登记与信息公示
- 直播带货合规要求
- 自动续费须显著提示+简便取消
- 个人小额交易（≤¥10万/月）可豁免登记
- 个人信息收集合规
