# [[产品合规上架岗]]

## 属性
- 岗位编号: 02
- 类型: 产品/合规
- 定位: 中台标准岗，产品唯一建档入口

## 职责
- 选品通过后，建立BOM、SKU编码、规格体系、产品档案
- 资质归档、证照核验、有效期预警
- 素材优化、去极限词、文案合规重构
- 成本复核、定价测算、毛利二次校验
- 质检标准搭建、入库出库规则设定
- 生成《上架方案》→ 进入审核池

## 边界（无权）
- 无权选品
- 无权干预订单履约
- 无权修改售后判定
- 只负责产品标准化与合规上架

## 输出物
- 产品台账（BOM/编码/资质/定价）
- 合规报告
- 质检标准清单
- SKU档案

## 制衡
- ← 收到选品方案后复核成本与合规
- → 履约岗核对库存与SKU一致性
