# [[资金支出审批单]]

## 属性
- 类型: 概念 | 资金管控
- 来源: 2026-06-29 Hans 定稿

## 标准模板（9字段，缺一不可）

| 序号 | 字段 | 说明 |
|:---:|:-----|:-----|
| 1 | 单据编号、生成日期 | 唯一标识 |
| 2 | 支出分类 | 投放/备货/售后赔付/平台服务费/合规外包/耗材 |
| 3 | 对应商品/项目、执行周期 | 关联SKU或项目 |
| 4 | 方案A/B/C三档预算 | 低/中/高三档投入测算 |
| 5 | 全成本测算 | 含预期毛利、ROI、资金占用时长 |
| 6 | 潜在风险 | 亏损风险/库存积压/客诉上升 |
| 7 | 岗位建议 | 建议通过/驳回/调整 |
| 8 | **股东审批栏** | 通过/驳回/调整预算 + 备注 |
| 9 | 后置归档栏 | 实际消耗 + 财税对账匹配 |

## 生成触发条件
- 广告投放新建/增额 → 投放营销岗
- 批量备货/采购 → 选品岗/履约岗
- 超额赔付/和解补偿 → 风控岗
- 外包服务/耗材采购 → 上架岗/履约岗

## 双池分流机制
| 类型 | 频次 | 适用场景 |
|:----|:----|:--------|
| 📅 周度计划资金池 | 每周复盘集中审批 | 月度广告预算、大批量备货、年度服务费、季度耗材 |
| 🚨 每日应急资金池 | 每日审核池同步 | 突发赔付、紧急补货、临时投放追加、合规检测付费 |

## 差异化驳回冷却周期
| 类型 | 冷却期 | 示例 |
|:----|:------|:----|
| 长期规划类 | **7天**禁重复推送 | 备货/月度投放/外包服务 |
| 短期经营类 | **3天**禁推送 | 少量补货/小额活动 |
| 应急售后类 | **无冷却** | 打假和解/大额客诉赔付 |

- 校验：财税岗每日对账，超支/无审批自动告警
