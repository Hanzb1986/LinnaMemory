## 全局标识位置
- 系统顶部导航栏右上角：「执行主体：AI Agent-Linna运营机器人」
- 所有看板/报表/审批单底部水印：「数据生成/操作执行：Lina Agent」
- 人工录入区块：灰色底色+「人工手动录入，非Agent自动生成」

## 操作日志区分字段
每条记录新增 `operator_type` 字段：
- `agent`：AI机器人自动执行
- `human`：人工手动录入/审核/补录

支持一键筛选两类数据，审计可快速区分。
