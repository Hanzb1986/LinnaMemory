# [[产品管理]]

## 属性
- 类型: 产品/品类
- 模块归属: 电商产品体系
- 覆盖: 产品编码/BOM/库存/上架规范/品类管理/参数合规/定价/生命周期

## 产品编码体系（BOM系统）
```
产品编码（SKU码）→ 品类编码 → 供应商编码 → 批次号
    ↓              ↓          ↓            ↓
  规格参数       类目属性    货源溯源     生产/入库日期
```

### 产品档案结构
| 维度 | 内容 | 格式 |
|:----|:-----|:----:|
| 编码 | 唯一SKU/条码/ERP编码 | 字符串 |
| BOM | 产品组成清单（包装/配件/说明书） | 结构化清单 |
| 规格 | 材质/尺寸/重量/颜色/型号 | 参数表 |
| 资质 | 3C/质检/食品许可/品牌授权 | 文件+有效期 |
| 库存 | 可用/在途/锁定/安全库存 | 数值+阈值 |
| 定价 | 成本价/批发价/零售价/活动价 | 价格阶梯 |
| 生命周期 | 新品/在售/清仓/下架/停售 | 状态标记 |

## 产品生命周期管理
```
新品备案（BOM建立+SKU分配）
  → 资质审核（3C/质检/品牌授权）
  → 参数录入（规格/BOM/图文物料）
  → 定价审核（成本→售价→毛利率验证）
  → SKU发布（上架/库存同步）
  → 在售巡检（合规/库存/价格/评价）
  → 滞销预警（库存周转率<阈值）
  → 清仓/下架（出清→下架→归档）
```

## 核心产出
- 产品管理制度.docx（含BOM编码规范）
- 产品上架SOP（资质→参数→定价→发布）
- 全品类产品台账.xlsx（编码/BOM/库存/资质/价格）
- 产品合规风险清单
- 产品资质汇编档案
- 产品生命周期报告
