# [[数据流向规范]]

## 属性
- 类型: 概念 | 系统架构
- 来源: 2026-06-29 Hans 定稿
- 规则: 数据单向流转，禁止反向修改上游原始数据

## 核心约束
1. **单向只读**：下游角色只能读上游成品文件，不能反向修改上游任何原始台账
2. **唯一媒介**：标准化文件/审批单传递数据，不能直接调取未归档草稿
3. **隔离读取**：业务AI不可读完整财务流水；客服仅读商品基础信息；财税可全链路读但不可改
4. **原始数据仅财税岗统一归档**，其他角色仅引用副本
5. **出口文件强制加盖**角色标识+数据等级标签

## 全角色数据流

```
① 选品招商岗
   入口：竞品公开数据 / 供应商报价 / 复盘报表 / 风控质量评级 / 黑名单
   出口：市场分析报告 → 供应商评级 → 选品方案 → 黑名单更新 → 备货审批单
   流向：审批通过 → 上架岗唯一输入

② 产品合规上架岗
   入口：选品岗输出(审批版) / 产品资质原件 / 客服高频问题
   出口：产品台账 → 合规报告 → 质检清单 → SKU档案 → 外包审批单
   流向：SKU档案+台账 → 履约岗唯一商品基础数据

③ 订单履约供应链岗
   入口：上架岗SKU档案 / 质检标准 / 平台API订单 / 物流同步 / 退货质检记录
   出口：订单台账 → 库存台账 → 履约异常日志 → 备货审批单
   流向：订单+库存台账 → 财税岗每日对账核心源

④ 质量售后风控岗
   入口：履约岗台账 / 上架岗质检标准 / 客服工单+聊天 / 平台售后 / 财税赔付成本
   出口：售后台账 → 客诉报表 → 整改清单 → 巡检日志 → 赔付审批单(含客服附件) → 供应商评级
   流向：售后损耗+评级 → 财税核算+选品迭代

⑤ 投放营销岗
   入口：财税利润/ROI报表 / 选品新品清单 / 广告平台API / 活动规则 / 风控售后率
   出口：投放数据表 → 优化方案 → 投放审批单 → 活动记录
   流向：广告消耗台账 → 财税岗分摊至SKU算净利润

⑥ 财税数据复盘岗（数据中枢）
   入口：全部5岗输出台账 + 银行流水/发票/税务回执 + 股东上传报价/账单 + 客服归档
   出口：收支台账 → 利润报表 → 资金审批台账 → 月度报告 → 政务预警 → 对账差异 → 复盘报告
   流向：复盘+利润数据回流选品/投放/风控岗迭代

⑦ 人工客服岗
   入口：平台客户咨询/售后对话 / 上架岗SKU详情(仅查询)
   出口：客服台账 → 客户工单+聊天存档 → 高频问题清单
   流向：工单+聊天 → 风控岗赔付审批单强制附件；高频问题→上架岗优化详情

⑧ 股东终审决策人
   入口：全角色推送审核池 / 可查阅全目录档案
   外部输入：上传供应商报价/平台账单/缴税回执/检测报告（人工补充源）
   出口（仅两类）：审批意见(归档至审批台账) / 阈值配置修改记录
```
