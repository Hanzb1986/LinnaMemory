# [[公司_有限责任公司]]

## 属性
- 类型: 公司治理
- 设立: 工商登记 + 营业执照
- 注册资本: 认缴制（无最低限额，特殊行业除外）
- 股东责任: 有限责任（以认缴出资额为限）
- 法定代表人: 执行董事/经理/董事长担任

## 公司架构

```
股东（会）——最高权力机构
    │
董事会（执行董事）——决策机构
    │
总经理（经理）——日常经营管理
    │
  ├─ 业务部（电商运营）
  ├─ 人事行政部（人力）
  ├─ 财务部（税务/会计）
  └─ 合规法务（法律顾问）
```

## 核心义务
- 工商年报：每年1月1日-6月30日
- 税务申报：月度/季度/年度
- 社保缴纳：员工入职30日内办理
- 会计账簿：依法设置，保存≥10年
