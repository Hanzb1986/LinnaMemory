# [[法规_著作权法]]

## 属性
- 类型: 法规 | 知识产权
- 索引: flk.npc.gov.cn
- 生效: 1991-06-01（2020修正）
- 监管: 国家版权局

## 核心要点
- 商品图片/文案/短视频受著作权保护
- 不得盗用他人店铺图片、详情文案
- AI生成内容的著作权归属待法律明确

## 应用场景
- 主图设计：使用原创图片或合法授权图片
- 商品文案：禁止抄袭同行描述
- 详情页视频：使用自有素材

## 处罚
- 侵权赔偿¥500~¥500万
- 情节严重可刑事追诉
