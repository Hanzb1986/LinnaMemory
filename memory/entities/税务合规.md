# [[税务合规]]

## 属性
- 类型: 税费/财务
- 适用法律: 税收征收管理法 + 增值税法 + 印花税法 + 企业所得税法 + 个人所得税法

## 电商常见税种

| 税种 | 税率 | 申报周期 | 说明 |
|:----|:----|:--------|:-----|
| 增值税（小规模） | 3%（2023-2027减按1%） | 季度 | 电商/toutiao店铺适用 |
| 增值税（一般纳税人） | 6%/9%/13% | 月度 | 年营收≥500万 |
| 企业所得税 | 25%（小微企业减按20%×25%） | 季度预缴+年度汇算 | 利润≤300万享优惠 |
| 个人所得税（工资） | 3%-45%超额累进 | 月度 | 代扣代缴 |
| 印花税 | 0.03%-0.1% | 按次/季度 | 购销合同 |
| 城市建设维护税 | 增值税×7%/5%/1% | 随增值税 | 市区7% |

## 关键合规红线
- 虚开发票→刑法第205条：虚开增值税专用发票罪（最高无期）
- 逃税→刑法第201条：逃避缴纳税款（3-7年）
- 税务登记：领取营业执照后30日内办理
