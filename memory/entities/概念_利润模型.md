# [[概念_利润模型]]

## 属性
- 类型: 概念 | 财务模型
- 状态: 待配置（等待平台参数+供应商数据）

## 利润公式

```
单品毛利润 = 售价 - 进价 - 物流费 - 平台佣金 - 包装耗材
单品毛利率 = 单品毛利润 / 售价 × 100%
净利润率 = (总营收 - 总成本 - 退货损耗 - 广告费) / 总营收 × 100%
```

## 参数占位

| 变量 | 来源 | 状态 |
|:----|:----|:----|
| 售价 | AI定价策略 | ⏳ 待激活 |
| 进价 | 供应商报价 | ⏳ 待供应商确认 |
| 物流费 | 供应商到客户运费 | ⏳ 待供应商确认 |
| 平台佣金 | 平台类目费率 | ⏳ 待平台确认 |
| 包装耗材 | 供应商包装费 | ⏳ 待确认 |
| 退货损耗率 | 历史数据测算 | ⏳ 无数据 |
| 广告费 | 投放策略 | ⏳ 待激活 |
