## 订单生命周期处理标准
| 阶段 | 库存动作 | 营收动作 | 备注 |
|:----|:--------|:--------|:-----|
| ⏳ 待付款 | 锁库存 | 不计营收 | 超时自动释放库存 |
| 📦 已付款待发货 | 扣减可售库存 | 确认应收 | — |
| 🚚 已发货 | — | 计提佣金、运费险 | — |
| ↩️ 未发货仅退款 | 库存回滚 | 全额冲减营收、佣金 | — |
| 🔄 已发货退货退款 | 实物质检后分类处理 | 区分可售/报废 | 人工拆包质检 |
| ✂️ 部分退款 | — | 按比例分摊佣金和广告 | — |
| ✅ 完成/关闭 | 数据永久锁定 | 禁止修改 | — |

## 活动优惠分摊规则
- 跨店满减、店铺券、单品券：按单品售价权重分摊
- 预售定金：定金当期入账，尾款完成核算全部佣金
- 平台官方补贴：不计入商家成本
- 商家赠品、自主满减：成本计入单品损耗，优惠冲减单品收入

## 固定核算公式（唯一标准，不可改动）
```
单品实收收入 = 订单售价 − 店铺券 − 商家满减分摊
单品直接成本 = 进货成本 + 单件分摊快递运费
单品平台扣费 = 佣金 + 运费险分摊
单品营销分摊 = SKU分摊广告消耗 + 付费活动分摊
单品售后损耗 = 退款赔付 + 退货报废损失
────────────────────────────────────
单品真实净利润 = 实收收入 − 直接成本 − 平台扣费 − 营销分摊 − 售后损耗
```

## 三大毛利率强制分栏展示
| 层级 | 包含 | 用途 |
|:----|:-----|:----|
| 产品毛利率 | 货品+运费 | 选品判断 |
| 平台毛利润 | 扣除佣金运费险 | 运营分析 |
| 整体净利润 | 全成本 | 股东真实盈利基准 |

## 库存校验逻辑
```
理论库存 = 期初库存 + 采购入库 − 售出 − 退货报废
```
账面与理论差额→标记盘盈/盘亏→必须真人实物盘点→AI禁止自动抹平

## ROI与损耗统计
- 单品ROI = 单品净利润 ÷ 对应SKU广告花费
- 整体ROI = 周期总净利润 ÷ 周期广告总消耗
- 售后率 = 售后订单数 ÷ 总成交订单数
- 损耗率 = 报废货品成本 ÷ 周期采购总成本

## 约束
所有计算严格套用固定公式；遇到未覆盖特殊场景禁止自行估算，标记规则缺失推送股东确认后更新
