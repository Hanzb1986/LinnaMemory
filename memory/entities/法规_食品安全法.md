# [[法规_食品安全法]]

## 属性
- 类型: 法规 | 食品类目专用
- 索引: flk.npc.gov.cn
- 生效: 2015-10-01（2021修正）
- 监管: 国家市场监督管理总局

## 适用范围
- 食品销售（预包装/散装/生鲜/零食）
- 食品添加剂
- 食品相关产品

## 核心要求
- 食品经营许可证（预包装食品备案）
- 标签标识合规：生产日期、保质期、配料表、营养成分表
- 特殊食品：保健食品/婴配/特医需专项注册
- 网络食品销售：信息公示、可追溯

## 处罚
- 货值金额10倍~15倍罚款
- 严重者终身禁入食品行业
