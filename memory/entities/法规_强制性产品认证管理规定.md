# [[法规_强制性产品认证管理规定]]

## 属性
- 类型: 法规 | 产品认证
- 索引: samr.gov.cn
- 生效: 2020修订
- 监管: 国家市场监督管理总局 / 认监委

## 核心要求
- 3C认证目录内的产品必须取得3C认证
- 无3C认证的产品禁止生产、销售、进口
- 3C目录包括：电器、儿童用品、汽车零部件、安防产品等

## 应用场景
- 家电类选品：确认3C认证编号
- 儿童玩具/用品：确认3C认证
- 电子数码配件：确认3C认证
