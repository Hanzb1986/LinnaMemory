## 架构总览

```
风控Agent ─┬─ 监控调研Agent（爬虫隔离检查）
           ├─ 监控合规Agent（规则更新检查）
           ├─ 监控运营Agent（操作合规检查）
           └─ 独立判断（资金/行为/关联风险）
```

## 安全防线（4层）

| 层级 | 机制 | 说明 |
|:----|:-----|:----|
| 1️⃣ 权限隔离 | 各Agent权限独立 | 禁止自审自批 |
| 2️⃣ 合规先行 | 合规Agent审校 | 内容上线前置拦截 |
| 3️⃣ 熔断止损 | 五级+三级熔断 | 资金/库存/亏损/合规/账号/集群 |
| 4️⃣ 独立兜底 | 风控Agent | 可强制关停任何功能 |
