# 人机协同体系

> 店主（Hans）与AI（Linna）的分工与流转

---

## 三类工作模式

### 模式A：全自动（AI独立完成）
- 数据监控、报表生成
- 常规客服、物流同步
- 库存同步、定时上下架
- 新规A类自动整改
- 风险熔断执行
- 日志记录

### 模式B：AI方案→店主终审
- 上新品、改主图、改标题
- 调价方案、活动报名
- 广告新建、大额赔付
- B类新规配合作业

### 模式C：店主独立完成
- 资质办理、人脸核验
- 12315/工商投诉处理
- 税务申报、保证金补缴
- 账号风控处理

## 流转闭环
1. AI执行常规工作 → 无打扰
2. AI识别的待审事项 → 进入审核池
3. 店主每日5-15分钟批量处理
4. 紧急风险优先熔断
