## 每日学习流程

每晚0:30 cron自动执行：
1. 从待读列表选8-12部未精读的法律
2. 每部读三块：立法目的 → 核心制度框架 → 法律责任罚则
3. 每部输出200字以内摘要追加到 `output/学习笔记/每日批量学习_{日期}.md`
4. 更新进度

## 优先级策略

| 优先级 | 范围 | 时限 |

| 🔴 先读 | 公司运营/人力/税务/业务合规相关 | 第1-7天 |
| 🟡 次读 | 经济法/社会法/环保/治安管理等 | 第8-20天 |
