# 从零开店全链路执行框架

> 严格区分**店主（Hans）必须操作事项** 与 **AI（Lina）全自动执行事项**
> 覆盖：资质办理→店铺入驻→供应链搭建→AI部署→日常运营

---

## 一、全链路阶段总览

| 阶段 | 店主工作量 | AI工作量 |
|:----|:---------|:---------|
| 🏗️ 筹备期 | 高（办执照/入驻/核验） | 辅助（查询/整理/出方案） |
| 🏪 搭建期 | 中（审核/确认/审批） | 高（生成方案/合规自检） |
| ⚙️ 运营期 | 低（日均5-15分钟） | 7×24全自动 |
| 🚀 扩展期 | 中（多店资质/资金统筹） | 高（隔离运营/风控巡检） |

## 二、筹备期店主操作清单（AI无代办权）

### 资质办理
1. 个体工商户注册：线上政务平台核名、提交设立、本人人脸核验
2. 经营范围必须包含「互联网销售」
3. 30日内税务信息确认、登记
4. 类目专项许可证（食品/美妆/3C等）
5. 本人名下收款银行卡
6. 刻公章

### 平台入驻
1. 注册商家账号，上传营业执照、身份证
2. 法人人脸核验、主体认证
3. 缴纳保证金（本人账户支付）
4. 填写店铺基础信息
5. 签署平台商家入驻协议

### 供应链搭建
1. 筛选供应商（1688工厂/代发渠道）
2. 敲定供货底价、发货时效、售后赔付规则
3. 确认代发隐私规则、物流渠道
4. 锁定供货底价区间，禁止AI低于底价售卖

## 三、人机审核超时机制（新增规则）

| 状态 | 操作 |
|:----|:----|
| 店主24小时内未处理 | 每6小时重复告警 |
| 店主48小时内未处理 | 自动锁定对应商品/广告操作，防止损失扩大 |

## 四、整体部署指令（10条）

1. 抓取全套电商法规+平台规则，输出对比报告，仅提供方案不替代法人操作
2. 自动生成店铺素材方案，前置法规自检，高风险操作存入待审核池
3. 7×24完成订单履约/客服/报表/库存/风控等白名单自动化
4. 搭建五级熔断机制，触发自动止损并告警
5. **A类规则自动批量整改**；**B类规则暂停业务+输出整改步骤+持续推送**
6. 严格执行黑白名单，黑名单无店主审核禁止执行
7. 完整落地时任务体系（小时/日/周/月/实时触发）
8. **全链路留存审计日志**（法规抓取/操作/合规整改），**合规判断溯源官方原文，杜绝AI幻觉**
9. 多店拓展兼容：按执照主体分组隔离运营
10. 所有经营法律、资金、处罚责任归属店主，AI仅执行/预警/出方案
