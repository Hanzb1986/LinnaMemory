# 法规采集风险约束与合规规则

> 整合自Hans 2026-06-29投喂的法规采集限制分析
> 核心：官方源访问限制、AI爬虫法律红线、标准化采集SOP

---

## 一、法规难找的6个核心原因

1. **法规分属多部门**：全国人大（法律）、国务院（行政法规）、市监总局/网信办（部门规章）、电商平台（自治规则）
2. **版本混乱**：旧版/修订版/废止版混杂，搜索引擎不做效力过滤
3. **关键词门槛**：官方标准名称与普通人搜索词不对应（"退换货规则"→《网络交易监督管理办法》）
4. **检索功能简陋**：政府网站不支持场景化模糊检索，无法按开店业务匹配法条
5. **部分不公开**：内部执法指引、平台风控细则、涉密法规无法获取
6. **第三方更新滞后**：免费法律网站内容残缺

## 二、爬虫抓取的硬性限制

| 限制 | 规则 |
|:----|:-----|
| 🔴 robots协议 | 所有政府官网限制高频批量抓取 |
| 🟡 频次限制 | 单IP ≤3次/分钟，单日总量封顶 |
| 🔴 禁止突破反爬 | 禁止代理池/验证码破解/多账号轮换（涉嫌刑法） |
| 🟢 允许使用范围 | 仅存储、内部AI学习、店铺合规自检；禁止对外售卖 |
| 🟢 数据留存要求 | 抓取须保留官网链接、时间、版本信息 |

## 三、授权抓取源（仅4个官方域名）

| 源 | URL | 覆盖范围 |
|:---|:----|:--------|
| 国家法律法规数据库 | https://flk.npc.gov.cn/ | 法律、行政法规 |
| 中国人大网 | http://www.npc.gov.cn/ | 法律原文、立法释义 |
| 中国政府网 | https://www.gov.cn/ | 行政法规、总局令 |
| 市监总局法规数据库 | https://sjfg.samr.gov.cn/ | 电商/广告/产品质量规章 |

## 四、标准化采集规则（合规Agent执行）

1. **单IP 60秒≤2次请求**，每日单站≤500条，夜间低峰执行
2. **隔离环境**：采集Agent与运营Agent完全物理隔离
3. **版本校验**：抓取后自动对比，仅留存现行有效版本
4. **禁止越权**：不模拟登录商家后台爬取平台规则，平台新规人工录入
5. **溯源存档**：每条法条存URL、发布日期、修订记录
6. **熔断机制**：触发验证码/IP封禁→立刻终止、次日重试

## 五、平台规则获取方式

平台公告（商家后台专属规则）不由AI抓取，使用以下方式：
- 人工定期登录商家后台，复制公告文本录入知识库
- 平台规则Agent仅做静态文件对比和更新提醒
