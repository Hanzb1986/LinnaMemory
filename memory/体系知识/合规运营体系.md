# 合规运营体系

> 法律法规 + 平台规则 + 店铺自检 三位一体

---

## 三位一体架构

```
     法律法规（国家层面）
          ↕
     平台规则（平台层面）
          ↕
    店铺自检（运营层面）
```

## 日常合规自检流程

1. **每日0点**：抓取法规更新 + 平台规则更新
2. **每日凌晨**：全店商品合规巡检
3. **上新预审**：匹配广告法/电商法/产品质量法
4. **客服话术**：消保法/民法典合规校验
5. **客诉处理**：自动匹配法条给出方案
6. **每周报告**：《店铺合规风险报告》

## 分级处置
- **A类（AI自修）**：禁词/标价/格式/话术
- **B类（店主配合）**：资质/法人/保证金/税务
