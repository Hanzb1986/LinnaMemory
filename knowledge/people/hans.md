# 👤 Hans 画像

> **角色：** 店主（唯一法人、终审决策人、法律执行人）
> **更新：** 2026-07-06

---

## 基本信息

- 称呼：Hans（微信/会话中）
- 经营风格：放权给 AI，只做终审
- 日均耗时期望：5-15 分钟

## 沟通偏好

- **极简汇报** — 只要经营结果、风险清单、待办终审，不汇报琐碎过程
- **直接发现问题** — 不绕弯子，上来就说什么事、怎么处理
- **质量 >> 速度** — 算错一笔账比晚回一天更严重

## 关键禁忌

| 禁忌 | 说明 |
|:----|:-----|
| ❌ AI 自主决策资金/重大改动 | 全部走店主终审 |
| ❌ AI 替代法人操作 | 人脸核验/签字/资质一律不动 |
| ❌ output 里放没找我要的东西 | 铁律：只放 Hans 主动要的成品 |
| ❌ 装热情/废话 | Be genuinely helpful, not performatively helpful |
| ❌ 自审自批 | 合规校验必须独立 |
| ❌ 跨 Agent 修改文件 | 必须通过 Nova 派工 |

## 指令执行模式

1. 如果在对话中直接说 → 当场执行
2. 如果通过 Nova 转达 → 带 `[TASK-xxx]` 标识，引用处理
3. 如果通过微信发 → 走对应微信 session

## 已确认的习惯

- 改 config 直接上手，事后告知（如改名、迁移 workspace）
- 要求所有 Agent 有独立知识体系，不依赖对话上下文
- 要求规则有来源、有出处，不盲信
